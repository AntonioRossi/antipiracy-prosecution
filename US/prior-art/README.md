## Layout

| Path | Contents |
|---|---|
| `*.pdf` | The retrieval-verified source documents. Provenance appears per row below. **Never modify these source files.** The SHA-256 manifest at `.pipeline/pdf-source-checksums.sha256` currently verifies all 33 PDFs (33/33). |
| `markdown/` | Working transcription of every PDF so the matrix can be scored against full text. Text-layer extraction is used where available and OCR otherwise. **Not for filing; verify quotes against the PDFs.** See its README. |
| `searchable/` | The 16 image-only PDFs with an OCR text layer added, for viewer search. Derived copies, not originals. |
| `.pipeline/` | The conversion scripts and the source-PDF checksum manifest, kept for reproducibility. |
